This is a personal project with several goals :

  • Help me manage my job search by keeping track of jobs statuses and applications (spoiler alert : there's a good chance this project will never do what I originally intended)
  • Learn as much and as fast as possible about Spring Boot, Hexagonal Architecture, Clean Architecture, MongoDB and Domain-Driven Design (DDD), and lots of other things I will forget (or not)

It is usable, although not to be trusted as a long-term tool. I will take any feedback though, and be glad should anyone find it useful.

Please feel free to contact me on .

Features

MyJobs allows you to :

  • Save interesting job offers
  • Add attachments, so that you can remember which CV or documents you sent to the recruiter
  • Manage jobs status : created, pending (i.e. an application has been sent), refused, accepted...
  • Add activities : keep track of actions you made
  • Receive reminders : define a delay to be reminded of jobs that have not been updated
  • Rate jobs : this can be used as a way to rate your interest for the job, the quality of the recruitment process. This is up to you !
